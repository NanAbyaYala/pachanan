# PACHAÑAN — Infraestructura Digital para el Conocimiento de Abya Yala

> Investigación completada 2026-05-31
> Nombre propuesto: PACHAÑAN = Pacha (tejido cósmico) + Ñan (camino)
> "El camino del tejido cósmico"

---

## 1. DIAGNÓSTICO: ¿Dónde estamos?

### Infraestructura física
- 80% de centros de datos del mundo en EE.UU. y Europa
- En Abya Yala: São Paulo, Santiago, Bogotá — todos operados por capital extranjero (AWS, Google, Microsoft)
- **Cero** centros de datos con gobernanza comunitaria o indígena
- Los pensadores andinos dependen de YouTube (Google), WhatsApp/Facebook (Meta)

### Lo que SÍ existe (para conectar, no empezar de cero)
- **IPFS**: red P2P de almacenamiento, 280K+ nodos, no depende de servidores centrales
- **Redes mesh comunitarias**: Rhizomatica (México), AlterMundi (Argentina), Red Fusa Libre (Colombia)
- **Mukurtu**: plataforma open source de archivo digital indígena con protocolos de acceso diferenciados
- **Local Contexts**: TK Labels (Traditional Knowledge) para marcar conocimiento con restricciones comunitarias
- **Wikipedias en quechua/aymara/guaraní/náhuatl**: existen pero son pequeñas

## 2. PROPUESTA: PACHAÑAN

### Principios
1. **Red, no pirámide** — como el Qhapaq Ñan: caminos que conectan sin someter
2. **Validación comunitaria** — no "journals" indexados, sino pares comunitarios (yachaq, asambleas)
3. **Multilingüismo radical** — cada contenido existe en varias lenguas sin que una sea "la original"
4. **Cosmovivencia, no disciplinas** — no separar filosofía de agricultura, astronomía de medicina
5. **Acceso diferenciado** — no todo conocimiento es para todos (TK Labels)
6. **Calendario andino** como referencia temporal

### Ontología de datos (categorías andinas como estructura)

| Categoría | Función |
|-----------|---------|
| **PACHA** | Contexto espacio-temporal (no fecha gregoriana) |
| **AYNI** | Relación de reciprocidad del conocimiento |
| **YANANTIN** | Conocimientos complementarios |
| **YACHAY** | Tipo de saber (ritual, agrícola, astronómico...) |
| **KAMAQEN** | Fuente de vitalidad del conocimiento |
| **CH'IXI** | Tensiones internas (no se resuelven, se documentan) |
| **WAKA** | Dónde reside la potencia de este conocimiento |

### Sistema de validación (3 niveles)
1. **Ayllu**: validado por la comunidad de origen
2. **Yanantin**: revisión entre pares complementarios
3. **Ruray**: validado por la práctica (si funciona, es válido)
