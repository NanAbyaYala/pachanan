# PACHAÑAN — El camino del tejido cósmico

> **Pacha** (tejido cósmico: espacio-tiempo-materia) + **Ñan** (camino, vía, sendero)
>
> Conocimiento abierto de **Abya Yala** (tierra en plena madurez).
> 100% open source. Sin centro. Sin dueño. Sin costo.

---

## ¿Qué es esto?

Un **bien común digital** para el conocimiento de Abya Yala — el continente que el colonialismo llamó "América".

Así como Europa tiene su genealogía (Grecia → Roma → Cristianismo → Ilustración), Abya Yala tiene la suya:

**Pacha → Yanantin → Ayni → Qhapaq Ñan → Sumak Kawsay**

Este proyecto existe para documentar, visualizar y compartir ese conocimiento — en sus propios términos, en sus propias lenguas, desde sus propias categorías.

---

## Principios

1. **Abierto por defecto** — Todo público. Sin "contenido premium".
2. **Multilingüe desde el inicio** — Español, quechua, aymara, guaraní, inglés, portugués.
3. **Sin centro** — No hay servidor central, no hay dueño. Forkeable.
4. **Datos primero** — Los datos son el núcleo. Las visualizaciones son accesorias.
5. **Ayni digital** — Contribuir es reciprocidad. No hay "usuarios", hay contribuidores.
6. **Sin barreras técnicas** — Editar Markdown es suficiente para contribuir.
7. **Soberanía de datos** — Archivos JSON descargables. Sin dependencia de plataformas.

## Cómo contribuir

1. Haz fork del repositorio
2. Elige qué quieres aportar (datos, textos, traducciones, visualizaciones)
3. Crea tu contribución siguiendo las guías en `/docs`
4. Envía un Pull Request

**No necesitas saber programar.** Editar un archivo Markdown o JSON ya es contribuir.
